# 🧪 Clinical Laboratory Diagnostic Excellence Hub

A professional medical reference web application designed for Laboratory Technicians and Clinicians. This tool provides detailed clinical data on **Lipid Profiles** and **Cardiac Markers**, extracted from the MLT Handbook.

## 🚀 Live Demo
**Check out the app here:** [PASTE_YOUR_STREAMLIT_URL_HERE]

## ✨ Features
- **Lipid Profile Analysis:** Detailed breakdown of Total Cholesterol, Triglycerides, HDL, and LDL.
- **Cardiac Marker Guide:** Precise timing (initial rise, peak, and normalization) for cardiac enzymes (Troponin, CK-MB, AST, LDH).
- **Clinical Interpretation:** Explanation of elevated and decreased causes for every test.
- **Specimen Requirements:** Clear instructions on tube types and fasting requirements.
- **Risk Ratios:** Automatic reference for cardiovascular risk ratios.

## 🛠️ Built With
- **Python**: The core logic.
- **Streamlit**: For the interactive web interface.
- **Markdown**: For clean medical documentation formatting.
